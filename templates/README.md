# Templates

Use these templates when adding or refreshing solution profiles, setup guides, and capability pages.

| Template | Use for |
|---|---|
| [system-profile.md](system-profile.md) | A solution profile under `solutions/`. |
| [build-guide.md](build-guide.md) | A setup guide under `setup-guides/`. |
| [capability-page.md](capability-page.md) | A capability comparison under `capabilities/`. |

## Workflow

1. Copy the relevant template into `solutions/`, `setup-guides/`, or `capabilities/`.
2. Replace placeholders with source-backed claims.
3. Leave unsupported or unverified claims as `Unknown`.
4. Link the new page from README, the folder README, and comparison pages.
