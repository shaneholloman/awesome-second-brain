# Templates

Use the common profile first, then add one category-specific analysis section.

## Common Template

- [system-profile.md](system-profile.md): Required for every system profile.

## Usage

1. Copy `system-profile.md` into `systems/<system-name>.md`.
2. Fill the common profile fields.
3. Copy the relevant category-specific analysis section into the same system profile.
4. Fill the category-specific questions with source-backed research.
