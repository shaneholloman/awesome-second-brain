# Setup Guides

Setup guides explain how to get a working AI second-brain workflow with an existing solution. They are not from-scratch implementation tutorials.

Every guide includes:

- prerequisites
- setup time
- install or connection steps
- data-source setup
- agent connection
- verification checks
- known failure modes

| Guide | Best for | Setup time |
|---|---|---|
| [Membase](membase.md) | Fastest shared memory across agents | Official: under 5 min |
| [OpenHuman](openhuman.md) | Local-first desktop personal AI with auto-fetch integrations | Official: minutes |
| [GBrain](gbrain.md) | Local/self-hosted agent brain | Hands-on: 30 min+ |
| [Supermemory](supermemory.md) | Hosted memory API and MCP | Official: minutes |
| [Mem0/OpenMemory](mem0-openmemory.md) | Developer memory API, hosted or self-hosted | Official: minutes |
| [Zep/Graphiti](zep-graphiti.md) | Temporal graph memory for apps | Maintainer estimate: 30-60 min |
| [Cognee](cognee.md) | Knowledge graph memory through MCP | Official: minutes with Docker |
| [Khoj](khoj.md) | Personal AI over files and notes | Official: minutes |
| [Obsidian/Logseq + AI bridge](obsidian-logseq.md) | Local-first PKM with agent access | Hands-on: 30-90 min |
